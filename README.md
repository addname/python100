# Python 100 问 - 从入门到精通

欢迎来到《Python 100 问》！

## 什么是 Python？

Python 是一门现代、简洁、易读的编程语言，专注于**简洁性、可读性和强大的生态系统**。它通过优雅的语法设计，让开发者能够快速构建各种类型的应用程序，从简单的脚本到复杂的Web应用、数据科学项目、人工智能系统等。无论您是想学习编程、进行数据分析、开发Web应用，还是探索人工智能，Python 都是一个绝佳的选择。

## 为什么写这本书？

我们注意到，许多开发者在学习 Python 时会遇到各种问题，从基础语法到高级特性，从标准库使用到第三方框架应用，每个阶段都有其独特的挑战。

本书的初衷就是为了提供一条清晰、循序渐进的学习路径。通过"一问一答"的形式，我们将复杂的知识点分解成一个个独立、易于理解的小模块。我们希望读者可以通过这种方式，像完成任务清单一样，逐个攻克 Python 学习中的难点，最终建立起对这门语言全面而深刻的理解。

## 🛠️ 本地开发指南

如果你想在本地运行或构建这个网站，你需要安装 Python 和 pip。

### 环境要求
- Python 3.7+
- pip

### 安装步骤

1. **克隆仓库**
   ```bash
   git clone https://github.com/addname/python100.git
   cd python100
   ```

2. **安装依赖**
   ```bash
   pip install mkdocs mkdocs-material mkdocs-git-revision-date-localized-plugin
   ```

3. **启动开发服务器**
   ```bash
   mkdocs serve
   ```

4. **访问网站**
   在浏览器中打开 `http://127.0.0.1:8000`

### 构建静态网站
```bash
mkdocs build
```
